# Hotel Booking API (Лабораторная работа)

## О проекте
Это REST API для управления отелями. Приложение позволяет добавлять, искать, обновлять и удалять информацию об отелях. Данные хранятся в памяти компьютера.

## Технологии
- Java 17
- Spring Boot
- Maven
- Lombok
- MapStruct
- Postman (тестирование)

## Архитектура
Проект состоит из трёх слоёв:
- **Controller** — принимает запросы от пользователя
- **Service** — содержит логику приложения
- **Repository** — хранит данные (в памяти)

## Модель данных (Hotel)
- id — номер отеля
- name — название
- address — адрес
- city — город
- country — страна
- stars — звёзды (1–5)
- description — описание
- pricePerNight — цена за ночь
- available — доступен ли

## Что умеет API

### GET запросы
- `/api/hotels` — показать все отели
- `/api/hotels/1` — показать отель с номером 1
- `/api/hotels?city=Москва` — показать отели в Москве
- `/api/hotels?stars=5` — показать 5-звёздочные отели
- `/api/hotels?city=Москва&stars=5` — показать 5-звёздочные отели в Москве

### Другие запросы
- **POST** /api/hotels — добавить отель
- **PUT** /api/hotels/1 — обновить отель
- **DELETE** /api/hotels/1 — удалить отель
